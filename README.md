## Like catch-mind game

Catch-mind is casual game. It is a picture quiz game in which Kungya characters appear as avatars. [read more](https://namu.wiki/w/%EC%BA%90%EC%B9%98%EB%A7%88%EC%9D%B8%EB%93%9C)

```bash

# watch mode
$ yarn start:dev

```
